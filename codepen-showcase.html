⚙️ Configuration Parameters

Customize the carousel behavior and appearance with these URL parameters:

Parameter Type Default Description Example
domain* string - Target domain for reviews www.google.com
theme light|dark|custom light Visual appearance theme dark
autoHeight boolean false Auto-adjust iframe height true
autoplay boolean true Enable automatic sliding false
interval number 5000 Autoplay delay (milliseconds) 8000
maxReviews number 10 Maximum reviews to display 5
minRating 1-5 1 Minimum star rating filter 4
showDate boolean true Display review dates false
showRating boolean true Display star ratings false

🚀 Quick Integration Guide

Get up and running in minutes with these simple steps:

1

Choose Your Domain

Replace www.google.com with your target domain in the iframe src URL.

2

Customize Parameters

Add URL parameters to customize theme, ratings, and behavior to match your needs.

3

Embed & Deploy

Copy the iframe code and paste it into your website. It works with all major platforms!
