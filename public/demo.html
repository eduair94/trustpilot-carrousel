üìö Quick Integration Guide

1. Basic Integration

Add the iframe to your website with the domain parameter:

<iframe src="https://your-domain.com/?domain=YOUR_DOMAIN" width="100%" height="400" frameborder="0"> </iframe>

2. Responsive Integration

For responsive design, wrap the iframe in a container:

<div style="position: relative; width: 100%; height: 400px;"> <iframe src="https://your-domain.com/?domain=YOUR_DOMAIN" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0"> </iframe> </div>

3. WordPress Integration

Add to your WordPress post or page:

[iframe src="https://your-domain.com/?domain=YOUR_DOMAIN" width="100%" height="400"]
